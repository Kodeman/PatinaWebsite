## Executive Summary

**Current State Analysis**: The existing About page at patina.cloud/about has solid structural bones but lacks the emotional depth, visual richness, and authentic storytelling that should define Patina. The page currently features placeholder imagery, fictional team members, and a generic startup timeline that doesn't communicate the real founding story or the "material feel" central to the brand.

## Design Direction: "The Workshop Wall"

### Conceptual Framework
Imagine walking into a master craftsperson's workshop. On one wall, there's a layered collage of meaningâ€”sketches pinned over photographs, material swatches tucked into frames, handwritten notes, worn tools displayed with pride. That's the feeling this page should evoke.

### Aesthetic Principles

1. **Layered Depth**: Elements should overlap, creating visual depth like fabric swatches stacked on a design table
2. **Material Presence**: Every section should feature tactile visual elementsâ€”wood grain, linen texture, leather patina
3. **Handcrafted Warmth**: Organic shapes, hand-drawn elements, warm photography with morning light
4. **Story Over Sell**: This is narrative-first, not conversion-first

### Visual Motifs
- **Wood Grain Textures**: Subtle backgrounds evoking craftsmanship
- **Fabric Swatches**: Section dividers that look like material samples
- **Aged Paper**: Quote blocks and pullout elements
- **Workshop Photography**: Real tools, hands at work, studio environments
- **The Strata Mark**: Used as a visual leitmotif throughout

## Technical Implementation Notes

### Animation Strategy

**Page Load Sequence**:
1. Hero background fades in (1.2s)
2. Strata Mark draws (0.8s, staggered lines)
3. Headline fades up (0.6s)
4. Subheadline fades up (0.4s delay)
5. Scroll indicator pulses (continuous)

**Scroll-Triggered Animations**:
- Sections fade in with 50px upward movement
- Value cards stagger in (0.1s delay each)
- Timeline entries alternate slide direction
- Quote blocks have subtle scale-up effect

**Hover States**:
- Cards lift with enhanced shadow
- Buttons have subtle color shift
- Images have gentle zoom (1.02 scale)

### Responsive Considerations

**Desktop (1200px+)**:
- Full visual experience with parallax
- Side-by-side layouts
- Large hero typography

**Tablet (768px-1199px)**:
- Stacked founder cards
- Single-column timeline
- Reduced parallax effects

**Mobile (< 768px)**:
- Full-width images
- Simplified animations
- Touch-friendly card interactions
- Collapsed timeline (vertical)

### Performance Optimization

- Lazy load images below fold
- Use WebP format with JPEG fallback
- Inline critical CSS for hero section
- Defer non-essential animations
- Compress textures to reasonable file sizes
