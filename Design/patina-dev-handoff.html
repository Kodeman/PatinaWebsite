Strata Mark System

The three horizontal lines that form our distinctive visual identity. Implement as a reusable component.

Size Variants

Compact (40px)
Medium (140px)
Large (200px)

Specifications Table

SizeBase WidthLine 1Line 2Line 3Gap
Large 200px 200×5px 160×4.25px 120×3.5px @ 50% 8px
Medium 140px 140×4px 112×3.4px 84×2.8px @ 50% 6px
Compact 40px 40×2px 32×1.7px 24×1.4px @ 50% 3px

Ratio Formula: Width = 100% / 80% / 60%. Height = 100% / 85% / 70%. Line 3 opacity = 50%. Border radius = height / 2.

React Component

StrataMark.tsx
interface StrataMarkProps {
  size: 'compact' | 'medium' | 'large';
  variant?: 'default' | 'reversed';
  align?: 'left' | 'center';
}

const sizes = {
  large: { base: 200, height: 5, gap: 8 },
  medium: { base: 140, height: 4, gap: 6 },
  compact: { base: 40, height: 2, gap: 3 },
};

export function StrataMark({ size, variant = 'default', align = 'center' }: StrataMarkProps) {
  const { base, height, gap } = sizes[size];
  
  const lines = [
    { width: base, height: height, opacity: 1 },
    { width: base * 0.8, height: height * 0.85, opacity: variant === 'reversed' ? 0.7 : 1 },
    { width: base * 0.6, height: height * 0.7, opacity: variant === 'reversed' ? 0.4 : 0.5 },
  ];
  
  return (
    <div className={`flex flex-col items-${align}`} style={{ gap }}>
      {lines.map((line, i) => (
        <span
          key={i}
          style={{
            width: line.width,
            height: line.height,
            backgroundColor: variant === 'reversed' 
              ? 'var(--patina-clay-beige)'
              : i === 0 ? 'var(--patina-mocha-brown)' : 'var(--patina-clay-beige)',
            opacity: line.opacity,
            borderRadius: line.height / 2,
          }}
        />
      ))}
    </div>
  );
}

Focus States

/* Focus ring for all interactive elements */
:focus-visible {
  outline: 2px solid var(--patina-clay-beige);
  outline-offset: 2px;
}

/* Remove default outline when using mouse */
:focus:not(:focus-visible) {
  outline: none;
}

Keyboard Navigation

  • All interactive elements reachable via Tab key
  • Logical focus order matches visual layout
  • Skip-to-content link as first focusable element
  • Escape key closes modals and dropdowns
  • Arrow keys navigate within menus and carousels
  • Enter/Space activate buttons and links
